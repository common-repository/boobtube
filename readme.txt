== Description ==

Boobtube is another YouTube video plugin with widget support.

== Installation ==

1. Upload boobtube director into /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Boobtube to a Widget in a sidebar for instance.
